# UI 布局说明

## 整体结构

项目包含两个 UI：

- `frontend`：Taro 微信小程序端，面向学生用户。
- `admin-frontend`：React/Vite 管理后台，面向管理员。

小程序统一使用 `PageShell`：

- 顶部固定自定义导航栏。
- 主内容区为浅灰背景、白色卡片。
- 主页面底部使用自定义 TabBar。
- 二级页面显示返回按钮，不显示底部 TabBar。

## 小程序底部 Tab

| Tab | 页面 | 功能 |
| --- | --- | --- |
| 首页 | `pages/index/index` | 今日课程、今日考试、天气、提醒订阅 |
| 课表 | `pages/schedule/index` | 周课表网格、学期/周选择、课程详情 |
| 成绩 | `pages/grades/index` | 成绩总览、考试安排、学期成绩表 |
| 个人 | `pages/profile/index` | 个人资料、同步、设置、消息、反馈入口 |

## 首页

文件：`frontend/src/pages/index/index.tsx`

- 顶部日期栏：日期、天气、同步状态。
- 操作区：订阅提醒按钮、日历图标。
- 今日安排：
  - 课程卡片：课程名、时间、节次、地点、教师、状态。
  - 考试卡片：考试名、时间、地点、座位、状态。
- 支持课程/考试折叠与展开。
- 未登录时显示绑定账号引导。
- 无安排时显示空状态。

## 课表页

文件：`frontend/src/pages/schedule/index.tsx`

- 顶部筛选栏：学期选择、周次选择。
- “回到今天”按钮：查看非本周时显示。
- 主体为课表网格：
  - 左侧显示节次和时间。
  - 顶部显示周一到周日及日期。
  - 课程块按星期和节次定位。
- 支持左右滑动切换周。
- 点击课程弹出详情：课程名、时间、教师、地点、周次。

## 成绩页

文件：`frontend/src/pages/grades/index.tsx`

- 学业总览卡片：总学分、平均分、绩点。
- 考试安排：
  - 默认显示下一场/当前考试摘要。
  - 展开后显示未考、待安排、已结束考试。
- 全部学期：
  - 每个学期一个卡片。
  - 显示课程数、学分、均分、绩点。
  - 展开后显示成绩表：课程、学分、成绩、绩点。
  - 低分成绩有特殊样式。

## 个人页

文件：`frontend/src/pages/profile/index.tsx`

- 个人资料卡：头像、姓名、学号、年级/层次、专业、学校。
- 点击资料卡打开编辑弹窗。
- 常用功能列表：
  - 通知
  - 意见反馈
  - 学校申请
  - 分享给好友
  - 关于
  - 同步
  - 设置
- 未绑定账号时显示账号绑定面板。

## 账号绑定

文件：`frontend/src/features/bind/BindAccountPanel.tsx`

- 学校搜索和选择。
- 教务账号密码输入。
- 首次导入协议确认。
- 可选保存密码并授权自动同步。
- 部分学校支持 WebView 登录同步。
- 绑定后同步课表、成绩、考试和个人资料。

## 二级页面

| 页面 | 功能 |
| --- | --- |
| 消息 | 查看平台通知、学校通知、个人消息，显示已读/未读 |
| 意见反馈 | 提交问题、建议和联系方式 |
| 学校接入申请 | 提交学校名称、教务系统网址、验证方式、联系方式、备注 |
| 设置 | 设置学期首周开始日期、退出登录、清理本机状态 |
| 关于 | 展示版本、安全与隐私说明、数据来源 |
| 网页登录同步 | 内嵌 WebView，接收网页侧抓取的课表/成绩/考试/资料数据 |

## 管理后台

文件：`admin-frontend/src/App.tsx`

布局：

- 左侧固定侧边栏导航。
- 顶部工具栏显示当前模块标题、描述和操作按钮。
- 主区域使用面板、表格、筛选栏和弹窗。
- 登录页输入 API 地址和 Admin Key。

主要模块：

- 概览：统计学校、用户、申请、反馈等数据。
- 学校管理：学校列表、启用/停用、学期开始时间、Provider 配置。
- 用户管理：用户筛选、查看学校和账号状态。
- 学校申请：按学校分组审核、通过/禁用/删除申请。
- 反馈管理：反馈列表和详情。
- 通知管理：创建全局/学校/用户通知，查看通知状态。
- 提醒配置：课程/考试提醒参数配置，支持 dry-run。

## 视觉风格

- 主色：蓝色 `#2f7bff` / `#2563eb`。
- 背景：浅灰 `#f7f8fb`。
- 卡片：白底、细边框、轻阴影。
- 小程序端偏移动卡片式布局。
- 管理后台偏密集表格和工具面板布局。
- 状态提示使用顶部浮层，错误为红色，普通提示为蓝/绿。
