# 后端服务器部署

### 服务器基础环境

```bash
sudo apt update
sudo apt install -y curl git nginx postgresql postgresql-contrib certbot python3-certbot-nginx
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs
sudo npm install -g pnpm pm2
```

### 创建数据库

```bash
sudo -u postgres psql
CREATE USER cschedule WITH PASSWORD '改成强密码';
CREATE DATABASE cschedule OWNER cschedule;
\q
```

### 拉取后端

```bash
sudo mkdir -p /opt/cschedule-backend
sudo chown -R ubuntu:ubuntu /opt/cschedule-backend
git clone --depth=1 https://github.com/leeyukiho/CSchedule-backend.git /opt/cschedule-backend
```

### 后端配置

```bash
cd /opt/cschedule-backend
cp .env.example .env
nano .env
```


### 后端首次部署

```bash
cd /opt/cschedule-backend
pnpm install
pnpm prisma generate
pnpm prisma migrate deploy
pnpm seed:schools
pnpm build
pm2 start "pnpm start" --name cschedule-backend
pm2 save
pm2 startup
```

### Nginx 配置

```bash
sudo nano /etc/nginx/sites-available/cschedule-backend
```

```nginx
server {
  listen 80;
  server_name api.unilinkcore.cn;

  location / {
    proxy_pass http://127.0.0.1:3000;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
  }
}
```

```bash
sudo ln -s /etc/nginx/sites-available/cschedule-backend /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

### HTTPS 证书

```bash
sudo certbot --nginx -d api.unilinkcore.cn
sudo certbot renew --dry-run
```

### 防火墙

```bash
sudo ufw allow OpenSSH
sudo ufw allow 'Nginx Full'
sudo ufw enable
```
