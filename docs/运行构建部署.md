# 运行构建部署

## 数据库初始化

```bash
cd backend
cp .env.example .env
pnpm install
pnpm prisma generate
pnpm prisma db push
pnpm seed:schools
```

## 数据库更新

```bash
cd backend
pnpm prisma db push
pnpm seed:schools
```

## 后端初始化

```bash
cd backend
pnpm install
pnpm prisma generate
pnpm build
pnpm dev
```

## 后端更新

```bash
cd backend
pnpm install
pnpm prisma generate
pnpm build
pnpm start
```

## 前端初始化

```bash
cd frontend
pnpm install
pnpm dev:weapp
```

## 前端更新

```bash
cd frontend
pnpm install
pnpm build:weapp
```
